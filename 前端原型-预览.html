快速排序算法 - Java实现

经典的快速排序算法实现，包含基准选择优化

Java 作者：zhangsan 2026-05-22
Java 复制代码
/** * 快速排序算法 - Java实现 * 时间复杂度: O(n log n) 平均, O(n²) 最坏 * 空间复杂度: O(log n) */ public class QuickSort { public static int[] quickSort(int[] arr, int left, int right) { if (left < right) { // 三数取中法选择基准 int pivotIndex = medianOfThree(arr, left, right); swap(arr, pivotIndex, right); int pivot = partition(arr, left, right); quickSort(arr, left, pivot - 1); quickSort(arr, pivot + 1, right); } return arr; } private static int partition(int[] arr, int left, int right) { int pivot = arr[right]; int i = left - 1; for (int j = left; j < right; j++) { if (arr[j] <= pivot) { i++; swap(arr, i, j); } } swap(arr, i + 1, right); return i + 1; } private static int medianOfThree(int[] arr, int left, int right) { int mid = left + (right - left) / 2; if (arr[left] > arr[mid]) swap(arr, left, mid); if (arr[left] > arr[right]) swap(arr, left, right); if (arr[mid] > arr[right]) swap(arr, mid, right); return mid; } private static void swap(int[] arr, int i, int j) { int temp = arr[i]; arr[i] = arr[j]; arr[j] = temp; } }

算法概述：这段代码实现了快速排序算法，使用了三数取中法优化基准选择。

时间复杂度：平均 O(n log n)，最坏 O(n²)（当数组已排序时）

空间复杂度：O(log n)（递归调用栈）

优化亮点：使用三数取中法（medianOfThree）选择基准，避免了最坏情况的发生概率。
