## 二、本地开发启动步骤

### 第1步：初始化数据库

打开MySQL命令行或客户端工具（如Navicat、DataGrip），执行：

```sql
-- 方式一：命令行
mysql -u root -p < aicodehub-backend/sql/init.sql

-- 方式二：在客户端工具中打开 init.sql 文件并执行
```

这会创建 `aicodehub` 数据库和所有表，并插入初始标签数据。

### 第2步：修改后端配置

打开 `aicodehub-backend/src/main/resources/application.yml`，修改数据库密码：

```yaml
spring:
  datasource:
    password: 你的MySQL密码  # ← 改成你自己的密码
```

### 第3步：启动后端

```bash
# 进入后端项目目录
cd aicodehub-backend

# 用Maven下载依赖并启动（第一次较慢，需要下载依赖）
mvn spring-boot:run
```

看到 `===== AiCodeHub 启动成功 =====` 表示后端启动成功。
API文档地址：http://localhost:8080/doc.html

### 第4步：启动前端

打开新的终端窗口：

```bash
# 进入前端项目目录
cd aicodehub-frontend

# 安装依赖（第一次需要）
npm install

# 启动开发服务器
npm run dev
```

看到 `Local: http://localhost:5173/` 表示前端启动成功。

### 第5步：访问网站

打开浏览器访问：http://localhost:5173

## 三、代码逐层解析

### 后端架构（请求处理流程）

```
用户请求 → Controller → Service → Mapper → MySQL
                                          ↓
用户响应 ← Controller ← Service ← Mapper ← MySQL

详细流程：
1. 前端发请求：POST /api/auth/login {username, password}
2. Controller 接收请求，调用 Service
3. Service 处理业务逻辑：
   - 用 AuthenticationManager 验证密码
   - 用 JwtUtil 生成 Token
   - 返回 LoginResponse
4. Mapper 操作数据库（MyBatis-Plus自动生成SQL）
5. 返回结果给前端
```

### 前端架构（数据流）

```
用户操作 → Vue组件 → API模块 → Axios → 后端接口
                                         ↓
页面更新 ← Vue响应式 ← Store ← Axios响应 ← 后端接口

详细流程：
1. 用户在 Login.vue 输入用户名密码，点击登录
2. Login.vue 调用 api/user.js 的 login() 方法
3. api/index.js 的 Axios 拦截器自动添加 Token
4. 请求到达后端 AuthController
5. 后端返回 JWT Token
6. api/index.js 的响应拦截器处理返回
7. Login.vue 调用 store/user.js 的 setLogin() 保存状态
8. Vue Router 跳转到 Dashboard 页面
```

### 核心代码解释

**后端 - SecurityConfig.java（安全配置核心）**
```java
// 哪些接口不需要登录就能访问
.requestMatchers("/api/auth/**").permitAll()    // 登录注册
.requestMatchers("/api/snippet/explore").permitAll()  // 探索广场

// 其他所有接口都需要登录
.anyRequest().authenticated()
```

**后端 - JwtUtil.java（JWT原理）**
```java
// 生成Token：把用户名和ID加密成一个字符串
String token = jwtUtil.generateToken(userId, username);

// 验证Token：解密字符串，取出用户名和ID
String username = jwtUtil.getUsernameFromToken(token);
```

**前端 - Axios拦截器（自动携带Token）**
```javascript
// 请求拦截器：每个请求自动带上Token
request.interceptors.request.use(config => {
  const token = localStorage.getItem('token')
  if (token) {
    config.headers.Authorization = `Bearer ${token}`
  }
  return config
})
```

**前端 - Vue 3 Composition API写法**
```javascript
// 响应式数据：数据变化 → 页面自动更新
const snippets = ref([])  // 声明响应式变量

// 生命周期：页面加载时自动调用
onMounted(() => { loadSnippets() })
```

## 四、部署上线步骤（简要）

### 1. 购买云服务器
- 阿里云/腾讯云，选 2核4G 配置
- 操作系统选 Ubuntu 22.04

### 2. 服务器环境安装
```bash
# 安装Docker
curl -fsSL https://get.docker.com | sh
sudo systemctl start docker

# 安装Docker Compose
sudo apt install docker-compose
```

### 3. 上传代码到服务器
```bash
# 用git或scp把整个项目上传到服务器
scp -r D:\wangzhan\wangzhan user@server:/home/user/aicodehub
```

### 4. 构建并启动
```bash
cd /home/user/aicodehub

# 先构建前端
cd aicodehub-frontend && npm install && npm run build && cd ..

# 一键启动所有服务
docker-compose up -d
```

### 5. 访问
服务器IP地址 → 就能访问你的网站了！
